# M+ feifei 品牌視覺策略分析報告

> **分析依據：** 16 張品牌素材（IG 限動截圖 4 張、行銷設計圖 9 張、產品實拍照 3 張）
> **分析日期：** 2026/03/06
> **用途：** 作為芳香噴霧品牌頁設計、WACA 商店視覺、小非人設的統一依據
> **備註：** 本報告為初稿，尚未納入短影音語言分析。待影片素材到齊後會補上語氣段落。

---

## 一、品牌定位聲明

**「用香氣為日常注入儀式感，讓每一次噴灑都是對自己的溫柔提醒。」**

M+ feifei 不是在賣香水，也不是在賣精油。她賣的是一種「用嗅覺完成情緒轉換」的微型儀式——噴一下，就像對自己說一句「我在。」這個定位介於美妝個護和身心靈之間，核心價值是**情緒療癒**和**自我關照**。

---

## 二、品牌個性關鍵字

**溫柔的 ╳ 有靈性的 ╳ 日常的 ╳ 女性力量的 ╳ 質感但不高冷的**

展開說明：
- **溫柔的：** 不說教、不強推，用「提醒」代替「改變」（「這一罐不是為了改變你，而是提醒你，你本來就已經很棒了」）
- **有靈性的：** 水晶能量、財神加持、磁場淨化，但不走重度宗教感，更偏向「生活中的小信仰」
- **日常的：** 強調「隨時隨地都可以使用」，不是特殊場合才用的奢侈品
- **女性力量的：** 「先愛自己」「為自己選擇當下最需要的能量」，核心是自我照顧
- **質感但不高冷的：** 包裝有設計感，但文案語氣親切，沒有距離感

## 三、目標受眾 Persona

### Persona A：小琪（核心客群）
- **年齡：** 28-38 歲
- **身份：** 上班族 / 小資創業者 / 年輕媽媽
- **生活型態：** 忙碌但渴望有品質的生活，會在睡前點蠟燭、週末去廟裡拜拜、關注星座運勢
- **消費習慣：** IG 是主要的資訊來源，看到喜歡的東西會先存起來，被限動打到會直接私訊問
- **情緒需求：** 覺得生活有點累，想要「被照顧」的感覺，但不想花太多錢或太多時間在自己身上
- **購買動機：** 「噴一下就能轉換心情」這件事的門檻很低，她願意試試看
- **對香霧的期待：** 不一定真的相信水晶能量，但喜歡這種「對自己好一點」的儀式感

### Persona B：雅婷（擴展客群）
- **年齡：** 22-30 歲
- **身份：** 身心靈愛好者，可能有在接觸塔羅、水晶、能量療癒
- **生活型態：** 會參加身心靈市集，IG 追蹤很多療癒系帳號，重視「能量」和「磁場」
- **消費習慣：** 對身心靈商品的價格接受度高，會因為產品的「能量故事」而購買
- **購買動機：** 水晶能量 + 財神加持對她來說有實質的吸引力，不只是儀式感
- **對香霧的期待：** 她會認真使用，當作日常能量工具，回購率最高

### 共同特徵
- 都是女性
- 都高度依賴 IG 做消費決策
- 都重視「感覺」多於「功能規格」
- 都會被「情緒化的文案」打動，而不是成分表
- 都有送禮需求（送閨蜜、送自己、送同事）

## 四、視覺語言分析

### 4.1 色彩體系

品牌已經建立了清晰的「一款一色」色彩系統，這是目前做得最好的部分：

| 產品 | 主色 | 建議 Hex | 輔色 | 情緒 |
|------|------|---------|------|------|
| 淨化 Purifying | 紫色 | #7B5EA7 | 薰衣草淡紫 #E8DFF5 | 寧靜、淨化、靈性 |
| 貴人 Attraction | 粉色 | #D4829C | 櫻花淡粉 #FDE8EF | 柔軟、魅力、溫暖 |
| 招財 Prosperity | 金色 | #C9A84C | 暖金淡色 #FFF5D6 | 豐盛、安定、力量 |
| 品牌通用 | 深棕 | #2C1810 | 琥珀瓶色 | 質感、天然、沉穩 |

**問題：** 行銷圖的背景色跳太多。招財款出現了紅色新年風和金色質感風兩種完全不同的氛圍，淨化款出現了泡泡漸層和水晶紫光兩種。建議統一每款產品的「背景語言」，不要因為檔期（如新年）就改變整個色調。

**建議：** 節慶素材可以加裝飾元素（燈籠、櫻花等），但背景主色調應該維持一致。例如招財款在新年檔期，背景可以是金色+紅色點綴，而不是整面紅色。

### 4.2 排版風格

**現有模式：**
- 產品圖偏右，文案偏左
- 字型偏手寫/書法感，混用標楷體、行書風格
- 標題大字、內文中字、成分小字的層次有做出來
- 「前調/中調/尾韻」的三段式描述是統一的結構

**問題：**
- 字型不統一。有些圖用偏正式的襯線體，有些用手寫風。同一系列的圖看起來像不同人做的。
- 文字量偏多。每張圖都塞了很多文字，在手機上看會覺得密。限動場景下用戶停留時間很短，應該更精簡。
- 背景處理方式跳動。有的用漸層、有的用實景、有的用素材堆疊。

**建議字型搭配：**
- 標題（產品名）：思源宋體 Bold 或類似的現代襯線體，保留東方質感但更乾淨
- 副標（英文名）：Playfair Display 或類似的優雅襯線英文體
- 內文：思源黑體 Regular，易讀、現代
- 點綴手寫字只用在金句 quote 上，不要用在產品資訊

### 4.3 影像風格

**做得好的：**
- 產品渲染圖的質感不錯（金色底座、水晶環繞、光暈效果），有高級感
- 三款產品各自的視覺氛圍有區隔（紫色水晶、粉色櫻花、金色元寶）

**需要改善的：**
- 產品實拍照（白底/淺色底那三張）偏素，跟行銷圖的質感落差大
- 如果要放到電商商品頁，建議重拍或用 AI 優化產品照的背景和光線
- QR Code + LINE 圖示放在右下角是統一的，但佔比有點大，視覺上有些干擾

### 4.4 一致性評估

| 項目 | 一致性 | 說明 |
|------|--------|------|
| 色彩系統（一款一色） | ✅ 高 | 三款色彩區分明確，包裝和行銷圖都有遵守 |
| 包裝設計 | ✅ 高 | 標籤框架統一，換色換圖示，辨識度好 |
| 產品照風格 | ⚠️ 中 | 渲染圖質感好，實拍照偏素，兩者落差大 |
| 行銷圖排版 | ⚠️ 低 | 每張圖的背景、字型、構圖方式都不太一樣 |
| 文案結構 | ✅ 高 | 三段式（前調/中調/尾韻）+ 能量故事，結構統一 |
| 文案語氣 | ✅ 高 | 溫柔、療癒、帶點靈性，語感一致 |
| 整體品牌感 | ⚠️ 中 | 有雛形但不夠系統化，需要一份視覺規範來收攏 |

## 五、文案語言特徵（從圖片文字萃取）

### 5.1 核心句式模式

feifei 的文案有幾個反覆出現的語言結構：

**「當你...，就會...」句式：**
- 「當你開始順流金錢的能量，財富也會更願意靠近你」
- 「當你開始喜歡自己的氣味，世界也會更容易靠近你」
- 「當世界過於喧囂，你需要一個能讓靈魂呼吸的出口」

**「不是...而是...」翻轉句式：**
- 「這一罐水晶香霧，不是為了改變你，而是提醒你，你本來就已經很棒了」
- 「斷捨離不只是物品，也包含磁場」
- 「買一罐送給自己，不是因為缺什麼，而是因為你值得被好好照顧」

**命令式但溫柔的：**
- 「噴一下，像對自己說一句：我在。」
- 「先愛自己，給未來的自己一點力量！」
- 「每天一噴，讓好事慢慢靠近你。」

**情境嵌入式：**
- 「店家開市前、重要會議前、約會前」
- 「可用於工作場合、求財運時刻」
- 「任何想提升自信、讓個人魅力 Upup 的時刻」

### 5.2 品牌金句（已驗證的高記憶點句子）

以下是從素材中出現頻率最高、且語感最強的句子，建議直接作為品牌資產保留：

1. 「你的氣場，決定你吸引的是什麼。」
2. 「每天一噴，讓好事慢慢靠近你。」
3. 「噴一下，像對自己說一句：我在。」
4. 「先愛自己，給未來的自己一點力量！」
5. 「有時候，你不是真的需要很多人，你只是需要好好站在自己那一邊。」

### 5.3 品牌語氣指南

| 維度 | 描述 | 範例 |
|------|------|------|
| 溫度 | 溫暖但不黏膩，像閨蜜不像老師 | ✅「讓好事慢慢靠近你」 ❌「這個產品可以改善你的運勢」 |
| 靈性感 | 有，但輕巧地帶過，不深入教義 | ✅「經過財神爺的加持」 ❌「此產品經由高僧開光」 |
| 自信度 | 柔性自信，不需要攻擊競品 | ✅「你本來就已經很棒了」 ❌「市面上的香水做不到這種效果」 |
| 銷售感 | 低壓銷售，用情緒帶動而非折扣促銷 | ✅「為自己選擇一個當下最需要的能量」 ❌「限時特價 599！」 |
| 稱呼 | 用「你」直接對話，不用「各位」「大家」 | ✅「你需要一個能讓靈魂呼吸的出口」 |

```

  H1（產品名中文）  思源宋體 Bold, 36-48px
  H2（產品名英文）  Playfair Display, 24-30px
  H3（段落標題）    思源黑體 Bold, 20-24px

  Caption（小字）   思源黑體 Light, 13-14px

間距：
  段落間距 1.6-1.8 倍行高
  區塊間距 40-60px
  留白要多不要少，讓畫面呼吸
```

```

  - 深色琥珀瓶身是視覺錨點，所有構圖以瓶身為中心
  - 背景用產品對應的淡色，不要太花
  - 光線偏暖，帶一點柔焦

行銷圖：
  - 每款產品限定一種背景風格，不要因檔期改變
    淨化：水晶 + 淡紫光暈
    貴人：櫻花 + 淡粉柔光
    招財：金色質感 + 暖光
  - 節慶檔期可以加裝飾元素，但不改變底色
  - 文字量控制在 50 字以內（限動用）

電商商品頁：
  - 主圖用渲染圖（有質感的那版）
  - 輔圖用產品三段式香調圖解
  - 不要用白底實拍素照，質感落差太大
```

## 七、Landing Page 設計指引

基於以上分析，芳香噴霧品牌頁的設計建議：

### 整體氛圍
- 背景色：米白 #FDF8F3，乾淨溫暖
- 不要暗色系（雖然行銷圖有用暗底，但長頁面閱讀暗底會壓迫）
- 每個產品區塊用各自的淡色做區隔
- 整體感覺：像走進一間安靜的選物店，溫柔但不華麗

### Hero Section
- 品牌金句：「你的氣場，決定你吸引的是什麼。」
- 副標：「三款水晶香霧，為生活注入剛剛好的能量」
- 視覺：三瓶並排或交錯排列

### 產品區塊
- 每款產品一個區塊，背景換對應淡色
- 左圖右文（手機版上圖下文）
- 資訊層級：產品名 → 一句話定位 → 香調三段描述 → 成分 → 價格 → CTA
- 不要把所有文案都放上去，精選最有力的句子

### CTA（行動呼籲）
- 語氣要跟品牌一致，不要用「立即購買」
- 建議：「為自己選一瓶」「把這份能量帶回家」
- 按鈕色用品牌對應色，不要用刺眼的紅色

## 八、小非人設語氣參考（初版）

基於品牌語氣分析，小非在群組裡的說話方式應該是：

### 跟品牌語氣一致的部分
- 溫暖、正面、輕靈性
- 用「你」不用「大家」
- 善用問句帶動互動（「最近有沒有覺得...」）
- 提到產品時用情境帶入，不硬推

### 小非自己的個性（跟 feifei 區隔）
- 比 feifei 更活潑、更可愛一點
- 偶爾會用 emoji，但不會每句都用
- 會讚美群友（「哇這個想法好棒！」）
- 會自嘲自己是 AI（「雖然我是小精靈，但我也想噴一下招財霧 😆」）
- 語句更短更口語，不會用太長的療癒系句子

### 小非不會做的事
- 不會用 feifei 的招牌金句（那是 feifei 的，不是小非的）
- 不會編造產品功效或水晶能量的說法
- 不會回答退換貨、客訴類問題（引導找 feifei）
- 不會過度頻繁推銷
