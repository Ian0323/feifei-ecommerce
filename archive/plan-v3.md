#### Part B：AI 品牌分析（Claude Code，1.5-2 小時）

素材到齊後，用 Claude Code 做兩件事：

```
⬜ 0.7 視覺素材 → 品牌 & 受眾策略

   輸入：業主提供的 10-20 張圖片素材
   
   Claude Code 分析內容：
   a) 受眾輪廓推斷
      - 從視覺風格推斷目標受眾的年齡層、生活型態、價值觀
      - 消費者的情緒需求（療癒？自信？儀式感？）
      - 跟同類型品牌的受眾比較
   
   b) 現有視覺語言分析
      - 主要色系和配色模式
      - 排版風格（字型、構圖、留白）
      - 影像風格（濾鏡、光線、場景偏好）
      - 情緒調性（溫暖/神秘/清新/華麗）
      - 一致性評估：哪些元素統一，哪些不統一
   
   c) 品牌策略產出
      - 品牌定位聲明（一句話）
      - 品牌個性關鍵字（3-5 個形容詞）
      - 視覺規範建議：
        - 主色 / 輔色 / 強調色（含 hex code）
        - 建議字型搭配
        - 影像風格指引
      - 受眾 Persona × 2（具體人物描述）
      - 品牌語氣指南（用於 landing page 文案和小非人設）
   
   輸出：docs/brand-strategy.md
   → 作為 Session 2 landing page 的設計依據
   → 作為 Session 1 小非 system prompt 的語氣參考

⬜ 0.8 短影音轉錄 → 語言風格萃取

   步驟一：語音轉文字
   - 工具：Whisper（本地跑，免費，Claude Code 可執行）
   - 輸入：5-10 支短影音的音檔
   - 輸出：每支影片的逐字稿 .txt
   
   步驟二：語言特徵分析（Claude Code）
   - 輸入：所有逐字稿
   - 分析項目：
     a) 常用詞彙和口頭禪（例：「其實」「真的」「對不對」）
     b) 語氣詞使用模式（～、！、？、emoji 偏好）
     c) 句子長度和節奏（短句多還是長句多）
     d) 開場方式（怎麼打招呼、怎麼切入主題）
     e) 產品帶入技巧（怎麼從聊天自然轉到推薦）
     f) 跟粉絲互動的方式（稱呼、問句比例、回應風格）
     g) 情感表達特徵（正面/柔和/直接/撒嬌）
   
   輸出兩份東西：
   1. docs/voice-analysis.md — feifei 的語言風格報告
   2. line-bot/data/system-prompt.md 的語氣段落
      - 「品牌整體語氣」→ landing page 文案 + 小非基底
      - 「小非專屬個性」→ 在品牌語氣上疊加差異化
        （比 feifei 更活潑一點、更可愛一點、但用詞習慣接近）
```

### Session 1｜「小非」AI 群組助理（練習價值最高，優先做）
**預計時間：** 4-5 小時（Weekend 1）
**目標：** 小非在導購群組上線，能自然互動＋自動回覆產品問題

#### 「小非」是什麼

小非是非非選品的 AI 小精靈，以 LINE 官方帳號身份加入導購群組。
群友都知道她是 AI，但她的語氣自然、會拋話題、會回應互動，讓群組保持活絡。
feifei 本人繼續在群組裡當主理人角色，小非是輔助，不搶戲份。

#### 核心架構

```
LINE 導購群組
      │
      ▼
 群組訊息進來（Webhook 接收）
      │
      ▼
 ┌─ 是否要回覆？（活躍度控制器）
 │
 ├─ 被 @小非 → 100% 回覆
 ├─ 產品關鍵字命中 → 100% 回覆
 ├─ 一般聊天訊息 → 20-30% 機率回覆
 └─ 冷卻中 / 深夜靜音 → 不回覆
      │
      ▼
  要回覆 → 打 Claude API
  ├── System Prompt（小非人設＋產品知識庫＋語氣設定）
  ├── 近 10 則群組訊息作為 context
  └── 回覆限制在 1-3 句（群組裡不講長篇大論）
      │
      ▼
  延遲 30-90 秒（隨機，模擬打字節奏）
      │
      ▼
  發送到群組

  ＋

  排程主動拋話題（每天 2-3 則）
  ├── 從話題庫隨機抽取
  ├── 時段：下午 2 點 / 晚上 8 點 / 週末早上
  └── 話題風格：生活感、儀式感、輕鬆提問
```

#### 活躍度控制規則

| 規則 | 設定 | 可調整 |
|------|------|--------|
| 被 @ 或產品關鍵字 | 必回 | — |
| 一般聊天 | 20-30% 機率回覆 | ✅ 可調 |
| 每小時主動發言上限 | 2-3 則 | ✅ 可調 |
| 連續回覆後冷卻 | 連回 3 則 → 靜默 30 分鐘 | ✅ 可調 |
| 深夜靜音 | 23:00-08:00 不發言 | ✅ 可調 |
| 主動拋話題排程 | 每天 2-3 則，固定時段 | ✅ 可調 |

## 風險與對策

| 風險 | 影響 | 對策 |
|------|------|------|
| PayNow 審核不通過或延遲 | WACA 無法線上收款 | 先用轉帳匯款上線，金流後補 |
| 水晶香霧被金流商歸類為化妝品 | 需要額外證明文件 | 提前確認產品分類，備好相關文件 |
| 業主素材遲交 | Session 被迫延期 | Session 0 就催齊，設定明確截止日 |
| WACA 免費版功能不足 | 某些活動功能受限 | 先用免費版驗證，不夠再升級輕量版（月費 $599） |
| 小非回覆品質不穩 | 說了奇怪的話影響品牌形象 | System prompt 嚴格約束 + 設定禁止話題清單 + 初期業主在群組裡觀察 |
| 小非太活躍被群友覺得煩 | 群友退群 | 活躍度參數保守起步（20%），觀察一週再調整 |
| Claude API 費用超預期 | 業主不想繼續付 | 初期先估算實際用量，設定每月 API 呼叫上限 |
| Railway 免費額度用完 | Bot 停機 | 監控用量，接近上限時考慮遷移到其他平台 or 升級 |
| 單一 session 超時 | 影響主線工作 | 超過預估 1.5 倍就先暫停，下次再處理 |
| 業主不會操作 WACA 圖片上傳 | 持續來找你處理 | Session 4 當面教＋錄螢幕影片＋寫 SOP |
