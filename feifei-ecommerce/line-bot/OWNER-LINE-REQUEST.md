# 業主 LINE 資料需求清單

> 用途：這份可直接傳給業主，用來收齊 `小非` 真實接線所需的 LINE 資料與權限。

## 一、最優先需要的資料

要把 `小非` 真正接到 LINE 官方帳號，最優先需要這些：

1. `LINE 官方帳號是否已開通`
2. `LINE Developers Console` 是否可登入
3. `Channel secret`
4. `Channel access token`
5. `是否已有導購群組可測試`

## 二、需要業主協助確認的資訊

請協助提供或確認：

1. 官方帳號名稱
2. 官方帳號登入方式是否可授權使用
3. 是否已開通 Messaging API
4. 是否已有測試群組
5. 群組目前大概幾人
6. 最常被問的問題前 5-10 題

## 三、導流資訊

請確認之後小非要優先導到哪裡：

1. `Google 表單`
2. `香霧 landing page`
3. `WACA 商店`
4. `Instagram`

如果目前還沒全部準備好，至少先確認：

- 一對一下單主要導哪裡
- 群組詢問主要導哪裡

## 四、群組互動設定要先確認

請先確認這幾件事：

1. 小非要不要進導購群組
2. 群組裡是否接受 AI 助理身份公開
3. 小非的語氣偏好
   - 活潑可愛
   - 溫柔療癒
   - 比較像小幫手
4. 有沒有不能碰的話題

## 五、可直接傳給業主的 LINE 訊息版

```text
我這邊要把小非正式接到 LINE，所以目前最優先需要妳幫我確認這幾件事：

1. LINE 官方帳號是不是已經開好了
2. LINE Developers Console 能不能登入
3. Channel secret
4. Channel access token
5. 有沒有一個可以先測試的小群組

另外也想一起確認：
- 小非之後主要是要導到表單、香霧頁還是商店
- 群組裡能不能接受小非是公開的 AI 助理身份
- 妳希望小非的感覺比較活潑、溫柔，還是偏小幫手型
- 最常被問的問題大概有哪些

如果妳現在還沒全部整理好也沒關係，最先給我：
官方帳號權限 + token/secret + 測試群組
這樣我就可以先把真實測試接起來。
```
