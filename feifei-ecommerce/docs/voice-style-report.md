# Voice Style Report

> 分析來源：`brand-analysis/video/794311416.549131.txt`、`brand-analysis/video/794311416.590203.txt`  
> 用途：作為 landing page 與 LINE「小非」文案調整依據。

## 1. 總結判斷

業主的說話風格不是冷靜的品牌文案型，也不是高冷的身心靈老師型。  
她比較像：

- 很有現場感的帶貨主理人
- 熱情、口語、反應直接
- 會把產品放進自己日常使用情境裡講
- 會自然把 `淨化 -> 貴人 -> 招財` 串成一套邏輯
- 能量、加持、財運、做生意、人緣，都是她很自然會提的語言

如果用一句話形容：

`她像是在跟熟客分享自己真的愛用的小寶貝，不是在念品牌稿。`

## 2. 語言特徵

### 2.1 強烈口語感

她大量使用：

- `今天`
- `我覺得`
- `我很喜歡`
- `真的`
- `然後`
- `就是`
- `好不好`
- `趕快`

這代表她的語氣不是修飾型，而是邊想邊講、邊感受邊分享。

### 2.2 情緒很滿

她會直接說：

- `超喜歡`
- `很緊張`
- `全身很麻`
- `很可愛`
- `money money money`
- `趕快買`

她不是只描述產品，而是把自己的情緒一起放進去，所以聽起來有感染力。

### 2.3 喜歡用生活動作描述產品

她不是抽象地講香氣，而是會說：

- 放在包包裡
- 到新的空間先噴一噴
- 保持空間乾淨
- 做生意需要貴人
- 開工、賺錢、招財

這表示她很適合用「情境導向」賣貨，而不是用規格導向。

### 2.4 很自然地混合開運與生活語言

她不會把產品講成宗教商品，但會很自然提到：

- 加持
- 過爐
- 武財神
- 貴人
- 招財
- 趨吉避兇

而且她講這些詞時不是嚴肅宗教口吻，而是很生活、很順口。

## 3. 銷售風格

### 3.1 先講感受，再講功能

她常先說：

- 我喜歡這個味道
- 這很可愛
- 這是我的小寶貝

然後才補：

- 這瓶是淨化
- 這瓶是貴人
- 這瓶是招財

### 3.2 很會做順序型導購

她有一個很清楚的銷售順序：

1. 先淨化
2. 再招貴人
3. 最後招財

這不是單品推銷，而是「一套三部曲」。

### 3.3 CTA 很直接

她不會繞很久，會直接說：

- 趕快
- 加一
- 去官方 LINE 下訂
- 去買

這代表頁面 CTA 可以比現在更直接一點。

## 4. 人設判斷

她的人設不是「高級質感香氛品牌創辦人」，而比較像：

- 有個人魅力的主理人
- 相信能量與開運，但不脫離生活
- 對做生意、貴人、財運、人緣有感
- 像在跟粉絲聊天，不像在上課

## 5. 對 landing page 的意義

目前 landing page 已有質感，但仍偏 `品牌頁`。  
根據這兩份逐字稿，更適合的方向應該是：

### 要加強的

- 主理人本人出場感
- 情境式選購邏輯
- `菲菲三部曲` 的概念
- 更口語的 CTA
- 更明顯的 `淨化 -> 貴人 -> 招財` 路徑

### 要避免的

- 太抽象的儀式感空話
- 太像文青品牌官網
- 太冷靜、太像設計稿

## 6. 建議文案方向

### Hero

不要只講「每天一噴，讓好事慢慢靠近你」。  
可以更像她本人，例如：

- `菲菲三部曲：淨化、貴人、招財`
- `這三瓶，是我自己真的很愛用的小寶貝`

### 產品介紹

每款要更像她本人在推薦：

- `如果我今天去很多地方，或覺得整個人很亂，我會先拿淨化。`
- `如果今天要見人，我就會拿貴人。`
- `如果今天要工作、開工、衝業績，我就會先噴招財。`

### CTA

可以更直接：

- `喜歡的話，直接到官方 LINE 下訂`
- `不知道怎麼選，就先來 LINE 問我`

## 7. 最終結論

頁面應該保留質感，但文案必須更像 `feifei 在帶貨`，而不是 `品牌在發聲明`。  
真正符合她市場定位的版本，應該是：

`有個人感、有開運語境、有生活情境、有直接導購。`
