# 業主資料需求清單

> 用途：這份清單可直接傳給業主，用來收齊 landing page 第一版上線所需資料。  
> 優先順序：先收 `必填資料`，其他可後補。

## 一、最優先資料

請先提供以下 4 類資料，這樣就可以把第一版香霧頁正式上線：

1. `三款香霧正式售價`
2. `三款產品正式圖片`
3. `Google 表單連結`
4. `LINE 官方帳號連結`

## 二、三款香霧資料

請針對以下三款產品，各提供一次完整資料：

- 淨化
- 貴人
- 招財

每款請提供：

1. `正式售價`
2. `一句話介紹`
3. `適合什麼時候使用`
4. `產品照片`
   - 至少 1 張主圖
   - 最好有 2 張：產品照 + 情境照

## 三、品牌資料

請提供以下品牌資訊：

1. `品牌名稱最終寫法`
2. `品牌故事短版`
   - 約 80-120 字
3. `希望第一次看到品牌的人記住什麼`
   - 一句話即可

## 四、下單流程資料

請提供以下資訊：

1. `Google 表單連結`
2. `匯款資訊是否已確認`
3. `出貨方式`
4. `大約出貨時間`

## 五、社群連結

請提供：

1. `LINE 官方帳號連結`
2. `Instagram 連結`
3. `是否要放其他連結`
   - 例如選物商店或童裝商店

## 六、可後補資料

這些不是第一版必填，但有的話頁面會更完整：

- 主理人照片
- 更多情境照
- 使用者回饋 / 見證
- 常見問題
- 每款產品更完整的推薦語

## 七、回傳格式建議

- 文字資料直接用 LINE 或文件回傳
- 圖片請用雲端資料夾整理
- 三款產品請分資料夾：
  - `淨化`
  - `貴人`
  - `招財`

## 八、可直接轉貼給業主的文字

```text
目前要先把香霧 landing page 第一版上線，所以我這邊最優先需要 4 類資料：

1. 三款香霧正式售價
2. 三款產品照片
3. Google 表單連結
4. LINE 官方帳號連結

另外如果方便，也請一起給我：
- 每款一句話介紹
- 什麼時候適合使用
- 品牌故事短版（80-120 字）
- 出貨方式與大概出貨時間

圖片可以直接分成「淨化 / 貴人 / 招財」三個資料夾給我。
```

```text
2. 三款產品照片

4. LINE 官方帳號連結

另外如果方便，也一起給我這些，頁面會更完整：

- 每款一句話介紹
- 每款適合什麼時候使用
- 品牌故事短版（大概 80-120 字就可以）
- 出貨方式
- 大概幾天出貨

圖片的部分最簡單就是分三個資料夾給我：
- 淨化

每款至少 1 張主圖，最好有 2 張：

```
