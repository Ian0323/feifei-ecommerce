業主 feifei（IG: myfeifei768）經營個人品牌，目前有三條產品線：

現況是 IG 短影音導流 → LINE 手動接單 → 表單＋匯款末五碼對帳 → 手動出貨。業主考慮請 Ian 幫忙做電商網站。

## 關鍵決策歷程

### 1. 商業模式判斷

業主走的是**個人品牌帶貨模式**，不是品牌電商模式。消費者買的是「feifei 推薦」，不是買某個品類。這決定了整個架構以「人」為中心，不是以「店」為中心。

### 2. 服務範圍定義

Ian 的條件：
- 這是 side project，主線是 MES 系統開發＋碩士論文
- 只用週末＋平日晚上空檔
- 盡量讓 Claude Code 完成 80%
- 一次性建置＋移交，不做長期維運
- 核心目的：累積 AI 應用實戰 portfolio

因此服務從「長期營運夥伴」砍成「一次性建置＋移交＋30 天保固」。

### 3. 業主的具體需求（溝通後確認）

1. SHOPLINE 童裝不動（抽成高但跟百貨綁定）
2. 接受表單＋匯款對帳的流程
3. 主要問題是 LINE 群組回覆不即時，客人會流失
4. 芳香噴霧和選物要**分開處理**
5. 芳香噴霧可以用一頁式網站（產品更動頻率低）
6. 選物需要有後台可以上下架和放活動
7. 希望 LINE 導購群組能有 AI 協助活絡氣氛

### 4. 架構決策

最終確定三件獨立的事：

| 項目 | 技術方案 | 金流 |
|------|----------|------|
| 芳香噴霧品牌頁 | 一頁式 HTML（Claude Code 刻）→ GitHub Pages | 無（Google 表單＋匯款） |
| 非非選物商店 | WACA 免費版 | PayNow（平台內建，免簽約） |
| LINE 群組助理「小非」 | LINE Messaging API + Claude API + node-cron | 無（導流至商店或表單） |

### 5. 平台選擇：為什麼選 WACA 不選 SHOPLINE

- SHOPLINE 最低方案 NT$1,500/月 + 開通費 $5,000 + 抽成 0.5-1%
- 業主已經被 SHOPLINE 抽成困擾
- WACA 免費版：$0 月費、不抽成、不綁約
- WACA 的 PayNow 金流免簽約、約 30 分鐘開通
- 缺點：圖片無法直接上傳（需用外部圖床）、後台介面不夠直覺
- 結論：以她的規模，WACA 免費版是零成本起步的最佳選擇

### 6. LINE「小非」的設計決策

**原始需求：** 業主想要有人在導購群組裡幫忙聊天活絡氣氛，Ian 想用 AI 代替人工。

**身份選擇：** 最初考慮讓 AI 假裝群友，但有兩個風險——被發現會毀信任、違反 LINE 使用條款。最終決定用**公開的 AI 助理身份**，取名「小非」，定位為非非選品的 AI 小精靈。

**技術架構：**
- LINE Webhook 接收群組訊息
- 活躍度控制器判斷是否回覆（被 @ 必回、關鍵字必回、一般訊息 20-30% 機率）
- 打 Claude API 生成回覆，帶入近 10 則訊息作為 context
- 延遲 30-90 秒發送（模擬自然節奏）
- node-cron 排程每天主動拋 2-3 則話題
- 深夜 23:00-08:00 靜音
- 連續回覆 3 則後冷卻 30 分鐘

### 7. AI 品牌分析的加入

為了解決「業主對品牌形象沒有明確方向」的問題，新增了兩個 AI 分析環節：

**視覺素材分析：** 收集業主過去的限動截圖、產品照、行銷圖 → Claude 分析色系、排版、風格、一致性 → 產出品牌策略報告。

**短影音語言萃取：** 收集 5-10 支短影音 → Whisper 轉錄 → Claude 分析語言特徵（口頭禪、句式、語氣詞）→ 產出語氣指南 + 小非 System Prompt 的語氣段落。

這兩個分析的產出會串聯到後續每個環節：landing page 的設計依據、WACA 的視覺風格、小非的說話方式。

### 8. 市場洞察：宮廟信眾市場

對話後期發現，香霧的核心客群其實是**台灣民間信仰的信眾**，特別是拜五路財神（五財公）的群體。這跟一開始假設的「身心靈生活風格族群」有很大差異：

- 信眾買的是「靈驗」，不是「儀式感」
- 信任鏈是「廟 → 人 → 產品」，不是 IG 演算法
- 年齡層更廣（25-65 歲），很多做小生意的老闆娘
- 對「有加持的東西」價格敏感度低

**建議的通路策略：** 線上（IG/品牌頁）走身心靈質感路線，線下和 LINE 群組走宮廟信眾路線。兩個世界用不同通路分開經營。

**對小非的影響：** 話題庫需要加入農民曆提醒、宮廟活動、拜拜知識、信眾見證等內容。

### 做得好的
- 一款一色的色彩系統
